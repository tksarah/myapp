# Todo アプリ 仕様書（最終版）

## 1. 概要

本アプリは、**ブラウザのみで動作する個人向け Todo 管理アプリ**である。  
サーバー通信を行わず、すべてのデータは **localStorage** に保存される。

- 対象環境：モダンブラウザ（Chrome / Edge / Firefox / Safari）
- 対応言語：日本語
- タイムゾーン：**日本時間（JST）基準**
- 外部ライブラリ：使用しない（Vanilla JS）

---

## 2. 基本設計方針

- **責務分離**
  - Todo と Category は独立したデータモデル
  - Todo は categoryId を参照するのみ
- **拡張前提設計**
  - フィルタ・ソート・UI 拡張を想定
- **データ互換性重視**
  - 既存データを壊さない設計
- **UX 優先**
  - クリック編集
  - 即時反映
  - 操作の一貫性

---

## 3. データ構造

### 3.1 Todo オブジェクト

```ts
Todo {
  id: string
  title: string
  categoryId: string
  completed: boolean
  dueDate: string | null   // "YYYY-MM-DD"
  createdAt: string        // ISO8601
  updatedAt: string        // ISO8601
}
```

### 3.2 Category オブジェクト

```ts
Category {
  id: string
  name: string
  color: string            // HEXカラーコード
  createdAt: string        // ISO8601
}
```

### 3.3 保存先

- Todos：`localStorage["todos"]`
- Categories：`localStorage["categories"]`

---

## 4. Todo 機能仕様

### 4.1 Todo 作成

- タイトル必須（空文字不可）
- カテゴリ選択必須（未分類あり）
- 期限は任意
- 作成時に createdAt / updatedAt を設定

### 4.2 Todo 表示

- 表示順（UI）  
  **チェックボックス → カテゴリ → タイトル → 期限 → 削除**
- カテゴリは **背景色付きタグ**として表示
- 期限は右寄せ・控えめ表示

### 4.3 完了管理

- チェックボックスで完了／未完了切替
- 完了 Todo は：
  - 透明度を下げる
  - タイトルに打ち消し線

### 4.4 Todo 編集（インライン）

| 項目 | 操作 |
|---|---|
| タイトル | クリック → Enter確定 / Escキャンセル |
| カテゴリ | クリック → セレクト変更 |
| 期限 | クリック → 日付入力 |

---

## 5. カテゴリ機能仕様

### 5.1 カテゴリ管理

- モーダル画面で管理
- 初期カテゴリ「未分類」は必ず存在
- 未分類は削除不可

### 5.2 カテゴリ追加

- 名前入力
- **ドット式カラーパレットから色選択**
- 選択色は category.color に保存

### 5.3 カテゴリ削除

- 削除時、該当 Todo は自動的に「未分類」へ移動

## 7. フィルタ機能仕様

### 7.1 状態フィルタ

- すべて
- 未完了
- 完了済み

### 7.2 カテゴリフィルタ

- 全カテゴリ
- 任意カテゴリ

### 7.3 期限フィルタ（JST基準）

| 種類 | 定義 |
|---|---|
| 今日 | 日本時間の当日 |
| 今週 | **月曜?日曜（JST）** |

※ UTC は使用しない

- 拡張容易（色・フィルタ・表示追加）
