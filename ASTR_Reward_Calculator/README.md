# ASTR Reward Tax Calculator

## 概要

Astar Network の dApp Staking で獲得したリワード（ASTR）から得られた利益を計算し、日本の所得税確定申告対応のためのデータ管理ツールです。

**特徴:**
- ✅ ブラウザのみで動作（サーバ通信なし）
- ✅ ローカルストレージにデータ自動保存
- ✅ CSVおよびJSON形式でのエクスポート対応
- ✅ レスポンシブデザイン（PC/タブレット/モバイル対応）
- ✅ リアルタイム集計計算

## 使い方

### 1. リワード情報を入力

- **📅 日付**: Claimした日付をカレンダーで選択
- **🪙 Claimされた量**: 獲得したASTRの量（小数点第8位まで対応）
- **💱 ドル円為替レート**: Claim時のレート（JPY/USD）
- **💲 ASTRレート**: Claim時のASTR価格（USD/ASTR）
- **⛽ トランザクションFee**: ガス代として支払ったASTRの量

### 2. 計算・追加

「➕ 追加」ボタンをクリックすると、自動的に以下が計算されます：
- ASTR量のドル換算金額
- ASTR量の円換算金額
- トランザクション費用

### 3. データ管理

**操作:**
- **編集**: テーブルの「編集」ボタンから再入力可能
- **削除**: テーブルの「削除」ボタンから削除
- **ソート**: 日付順に自動でソート（新しい順）

### 4. データのエクスポート

- **📥 CSVエクスポート**: Excelで開いて詳細分析が可能
- **💾 JSONエクスポート**: バックアップやリストア用
- **📂 JSONインポート**: 以前エクスポートしたデータを読み込み、既存データに追加できます（重複IDはスキップ）

---

## データ保存について

- ✅ すべてのデータはブラウザのローカルストレージに自動保存
- ✅ サーバへの送信なし（完全プライベート）
- ✅ ブラウザのキャッシュをクリアするとデータが失われるため、定期的にエクスポート推奨

## 計算ロジック

### 入力値

| 項目 | 値 |
|------|-----|
| ASTR量 | 50.12345678 |
| ドル円レート | 150.50 |
| ASTRレート | 0.1234 |
| Fee | 0.5 |

### 各Entryの計算結果

| 項目 | 計算式 | 結果 |
|------|--------|------|
| リワード（USD） | 50.12345678 × 0.1234 | $6.18 |
| リワード（JPY） | $6.18 × 150.50 | ¥930 |
| Fee相当額（USD） | 0.5 × 0.1234 | $0.06 |
| Fee相当額（JPY） | $0.06 × 150.50 | ¥9 |
| 利益（USD） | (50.12345678 - 0.5) × 0.1234 | $6.11 |
| 利益（JPY） | $6.11 × 150.50 | ¥919 |

### 集計計算

| 項目 | 計算式 | 結果 |
|------|--------|------|
| 総リワード（JPY） | Σ(各Entryのリワード（JPY）) | ¥930 |
| 総Fee（JPY） | Σ(各EntryのFee相当額（JPY）) | ¥9 |
| **最終損益額（JPY）** | 総リワード（JPY） - 総Fee（JPY） | ¥921 |

**※ 最終損益額が確定申告時の所得金額となります**

## 集計表示

画面右側の「📊 集計結果」に以下が表示されます：

- **Claim回数**: 登録したエントリの件数
- **合計ASTR量**: すべてのClaimの合計
- **合計Fee（ASTR）**: すべてのトランザクション費用のASTR合計
- **合計Fee（JPY）**: すべてのトランザクション費用の円換算合計
- **合計リワード（USD）**: すべてのリワードのUSD合計
- **合計リワード（JPY）**: すべてのリワードのJPY合計
- **💰 最終損益額（JPY）**: **確定申告用の所得金額**（総リワード - 総Fee）

## FAQ

### Q: ですべてのデータが失われた場合、復元できますか？

A: 定期的にJSONでエクスポートしておくことで、バックアップから復元可能です。

### Q: 複数のブラウザでデータを共有できますか？

A: 各ブラウザで独立したローカルストレージを持つため、データを共有するにはJSONファイルをインポートする必要があります。

### Q: 確定申告に使用できますか？

A: **このツールは参考情報です。必ず税理士または税務署にご相談ください。**

仮想通貨の所得計算は複雑です：
- Claim時点での所得認識
- FIFO / 移動平均法による取得原価計算
- 外貨建資産の為替差損益など
